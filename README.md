# GraceFeed

Share information of activities, events, articles, to congregation member

## Tech Stack

- **Platform**: Compose Multiplatform (Android + iOS)
- **Package**: `com.gracefeed`
- **Backend**: Firebase (Firestore + Auth)
- **UI**: Material 3 (Jetpack Compose)
- **DI**: Koin
- **Networking**: Ktor
- **Architecture**: MVI (Model-View-Intent) with Clean Architecture

## Features

- **Bulletin Feed** — Weekly digital bulletin replacing PDFs, featuring news, events, daily verse, and weekday gatherings
- **Service Schedule** — Display weekly Sunday service schedule with ability to show merged or modified services for special weeks
- **Event Calendar** — Browse church events (one-time and recurring) with RSVP option and category filtering
- **Sermon Archive** — Archive of Sunday sermons with YouTube video playback for all service types
- **Onboarding** — Welcome walkthrough for first-time users (3 pages)
- **Settings** — App preferences including dark mode toggle

## Getting Started

### Prerequisites

- Android Studio Hedgehog or later
- Xcode 15+ (for iOS)
- JDK 17+

### Build & Run

**Android:**
```bash
./gradlew :composeApp:assembleDebug
```

**iOS:**
```bash
cd iosApp
pod install
open iosApp.xcworkspace
```
